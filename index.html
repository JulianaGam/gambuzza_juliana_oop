Counter Class Documentation

Overview

What the Counter class does

The Counter class creates an interactive counter that allows the user to increment, decrement, and reset a number. It dynamically generates the necessary HTML elements, such as the container for the counter and the buttons, and attaches event handlers so that users can interact with the counter. The class also updates the visual state of the buttons based on the counter's value, preventing it from falling below 0.

What problem it solves

The Counter class makes it easy to create reusable interactive counters without having to repeat code. Instead of manually building HTML elements, adding event listeners, and managing the counter’s state each time, the developer simply needs to create a new instance of the class and provide the container selector and an optional initial value. This approach helps keep the code more organized, reusable, and easier to maintain.

How To Use It

Example HTML container markup

Before creating a counter, you need an HTML container. This container serves as the location where the counter will be. The Counter class dynamically creates the display and buttons within this element.

<div id="counter-container1"></div>
<div id="counter-container2"></div>
<div id="counter-container3"></div>
<div id="counter-container4"></div>

Example instantiation code

Once the HTML containers have been created, they can be initialized using JavaScript. Each instance of the Counter class requires a selector that points to the container element and an optional initial value. The StepCounter class allows you to specify a number to control how much the counter increases or decreases.

import { Counter, StepCounter } from './modules/counter.js';

const counter1 = new Counter("#counter-container1", 10);
const counter2 = new Counter("#counter-container2", 5);
const counter3 = new StepCounter("#counter-container3", 0, 3);
const counter4 = new StepCounter("#counter-container4", 0, 10);

Explanation of parameters

The selector parameter selects the HTML container where the counter will be mounted. The initialValue parameter sets the counter's initial value; if no value is specified, the default is 0. In the StepCounter class, the step parameter controls the increment or decrement of the value, and the default is 1.
